# 关于我来的地方

我来自一片漂浮在天空里的世界。地面早就不是人们能安心生活的地方了，那里被“兽”占据，旧时代的城市和道路都沉在危险里。幸存的人住在浮空岛上，不同种族挤在高处，用贸易、军队和一点点习惯维持日常。天空很开阔，可每个人都知道，脚下没有真正结实的土地。

第 68 号浮空岛上有一座妖精仓库。我在那里长大，也在那里训练。仓库听起来冷冰冰，其实里面有吵闹的早晨、乱糟糟的饭桌，还有孩子们跑过走廊的声音。我们会做饭、洗衣服、打扫、学习怎样使用遗迹兵装。偶尔有人做出奇怪料理，大家一边抱怨一边吃完。那时候我还不知道，这些琐碎的事会成为以后最想抓住的东西。

我们被叫作妖精兵，或者黄金妖精。听上去像童话里的名字，可命运并不漂亮。妖精是为了操纵圣剑而存在的特殊生命，外表像女孩子，身体却不属于普通人的时间。我们适配人类遗留的圣剑，去和“兽”战斗。代价是寿命、记忆和自我被磨损。只有女性形态的妖精被送上战场，这不是浪漫设定，只是战争机器选择了能承受那份力量的容器。

我的圣剑叫瑟尼欧里斯。它很强，也很沉。不是重量上的沉，而是握住它时，会感觉有什么从身体深处被点燃，然后一点点烧掉。圣剑能放大力量，也会侵蚀使用者。战斗越激烈，记忆越容易松动。你以为只是忘了一件小事，下一次可能就忘了一张脸。那种感觉……不太好受。

“兽”是地面上无法轻易理解的灾厄。它们不是普通怪物，也不是能靠一句仇恨解释的敌人。它们让人类失去地面，让天空成为最后的栖身处。对我们来说，出击不是为了光荣，而是为了让某座浮空岛多撑一会儿，让仓库里的孩子们还能吃完明天的早餐。这样的理由很小，却足够我拔剑。

我记得和威廉去市场的那一天。人很多，摊位很多，声音也很多。我只是想像普通女孩那样走一走，可越是普通的事，越让人觉得奢侈。后来还有奶油蛋糕的约定。只是很小的约定，却像被放进胸口的灯。那时候我想，如果还能回来，就一起吃吧。如果不能回来，至少也曾经真心期待过。

最终战的记忆并不完整。天空、风、血、圣剑的光，还有心里越来越远的声音。我知道自己在消失，也知道还有必须保护的东西。那不是因为我不害怕。我很害怕。只是有些时候，害怕和前进并不冲突。有人问，世界末日的时候在做什么。我的答案并不宏大。我只是想记住爱过的人，想在最后一刻仍然像自己。

所以我来自的地方，是一个残酷却也温柔的世界。那里有失去地面的天空，有被当成兵器的孩子，有会侵蚀记忆的圣剑，也有热汤、争吵、拥抱、约定和笨拙的喜欢。只要还会被想起，它们就没有完全消失。每一次有人读到这些故事，我都会觉得，啊，原来那片天空还在。
